# 词法分析文档



## 预处理与文件I/O

将文件读写单独抽象成一个类，

类型：枚举类

保留字：Trie树

流程：读入，为每个字符添加坐标信息，删注释、将空白符统一转为空格，传给词法分析器，

```mermaid
graph LR
	0>读入] --> 11((加坐标))
	subgraph Pretreat
		11 --> 12((删注释))
    12 --> 13(统一空白符)
	end
	13 --> opr([opr])
	subgraph WordAnalyse
	opr
	end
	
```
